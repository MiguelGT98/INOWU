Sonograph

Sistema de de gestión de clientes

Sonograph CRM es un sistema web desarrollado en HTML, CSS, JAVASCRIPT, PHP AND MYSQL .

El sistema ayuda a los empleados de Sonograph a saber cuando su trabajo sera completado. Por su parte los vendedores tienen un generador de trabajos, al momento de realizar uno, se le asigna a al diseñador con menos trabajo.

Funcionalidades Clave:

Peticiones PHP, Base de datos para los usuarios.

Dancerito

App para compra de eventos

Dancerito es una plataforma web/movil para comprar boletos.

Hecha utilizando flutter, esta app ayuda a los usuarios a comprar boletos para eventos de baile cerca de ellos. Esta app es completamente funcional en web. Donde se tiene una pasarela de pagos para la compra de boletos.

Funcionalidades Clave:

Base de datos, notificaciones, Inicio de sesión con redes sociales, pasarela de pagos.

CONACOMEE

App de conferencias, multiusos

CONACOMEE es un app para iOS y android que se conecta con un administrador Web

Esta app tiene una serie de funcionalidades. Sirve como conexión para los clientes de CONACOMEE para subir productos, cursos, revistas, eventos, entre muchos otros. Por otra parte tiene una seccion para las conferencias actuales que la CONACOMEE realiza.

Funcionalidades Clave:

Notificaciones, Base de datos, Admin Web, Cambios en tiempo real.

Paper Plus

E-Commerce

Paper Plus is an E-Commerce that was built for Web.

Este E-Commerce te permite personalizar los productos, con texto e imagenes. Cada cambio tiene un peso en el precio final. Como un E-Commerce puedes revisar tus compras y salvarlas para despue´s. Algo único es que se conecta con Fedex, para enviar los productos al usuario.

Funcionalidades Clave:

Base de datos, Conexión con fedex, Presonalización de productos.

Skybox

Este juego es una experiensa inmersiva, donde los sentidos y el juego interactuan para brindar una experiencia que te llevara a los limites. Los objetos del juego se pueden sentir en la realidad.

Wknbuy

Venta de Cupones

Wknbuy es una app desarrollada para iOS y Android con un administrador web.

En Wknbuy un administrador desde su computadora o teléfono puede registrar el precio, cantidad y puntos que una promocion tiene. Tambien puede determinar cuando abre o cierra una venta, por ciudad. El usuario puede comprar las varias promociones mediante pagos con tarjeta o con puntos.

Funcionalidades Clave:

Ventas basadas en horarios, uso de multimedia, notificaciones, base de datos, pasarela de pagos.

Mr. Pampas

Información del restaurante

Mr. Pampas es una app desarrollada para iOS y Android

Esta app ofrece una vista al menu del restaurante, e incluye la ubicación de todos los locales que hay en México junto con su información de contacto. La app tambien incluye una tarjeta de visitas. Cuando juntas tres visitas se te otorga un premio especial dictado desde la app.

Funcionalidades Clave:

Notificaciones, Google Maps, Tarjeta de cliente frecuente.

Wknfly

Reservaciones

Mr. Pampas es una app desarrollada para iOS y Android

En Wknfly, se resuelve el problema de que te salten en la fila. Te llaman desde la aplicación de hostess, donde te llega una notificación a tu telefono. Tambien si no quieres hacer la fila, cuenta con un plan para pagar un extra y ser el siguiente en la fila.

Base de datos en tiempo real, pasarela de pagos, aplicacion de hostess y de cliente, localización en tiempo real.
