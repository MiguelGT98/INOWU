Sonograph

Sistema de de gestión de clientes

Sonograph CRM es un sistema web desarrollado en HTML, CSS, JAVASCRIPT, PHP AND MYSQL .

Este sistema ayuda a los empleados de Sonograph a manejar el tiempo en el que sus trabajos asignados serán completados. Por otro lado, los vendedores cuentan con un generador de trabajos, el cual automáticamente asigna el proyecto a los diseñadores dependiendo la carga de trabajo que tengan.

Funcionalidades Clave:

Peticiones PHP, Base de datos para los usuarios.

Dancerito

App para compra de eventos

Dancerito es una plataforma web y móvil, la cual permite a los usuarios conocer los congresos de baile cercanos a su ubicación, así como comprar entradas para los mismos. Hecha utilizando flutter, esta app es completamente funcional en web, y cuenta con una pasarela de pagos para la compra de boletos.

Funcionalidades Clave:

Base de datos, notificaciones, Inicio de sesión con redes sociales, pasarela de pagos.

CONACOMEE

App de conferencias, multiusos

CONACOMEE es un app para iOS y Android que puede administrarse mediante un portal web.

Esta app tiene una gran variedad de funcionalidades: Sirve como conexión para los usuarios y socios de CONACOMEE para subir y visualizar productos, cursos, revistas, eventos, entre muchos otros; cuenta con una sección para las conferencias actuales que la CONACOMEE realiza; publicación de actividad de CONACOMEE.

Funcionalidades Clave:

Notificaciones, Base de datos, Admin Web, Cambios en tiempo real.

Evolution Fitness

Itinerario de comidas y ejercicio.

Evolution Fitness es un app para iOS y android que se conecta con un administrador web.

Funciona como un medio para llevar un registro e itinerario de comidas, así como planes de rutinas para usuarios del gimasio. Los usuarios pueden pagar su membresía y agendar seguimientos con el nutriólogo o entrenador. Todo esto es manejado por un administrador web, el cual personaliza los planes para los usuarios.

Funcionalidades Clave:

Administracion de multimedia, notificaciones, base de datos.

Paper Plus

E-Commerce

Este e-Commerce permite a sus usuarios personalizar productos con texto e imágenes de acuerdo a sus necesidades, lo cual va generando un cambio en el precio final de los productos a comprar. Cuenta con la función de revisar y guardar pedidos para revisar después. La página se conecta con FedEx para realizar los envíos de órdenes de los clientes.

Funcionalidades Clave:

Base de datos, Conexión con fedex, Presonalización de productos.

Skybox

Experiencia Multisensorial

Skybox es un juego de realidad virtual que funciona con el HTC Vive.

Este juego es una experiensa inmersiva, donde los sentidos y el juego interactúan para brindar al usuario una experiencia que lo llevará a los limites. Los objetos del juego se pueden sentir en la realidad.

Funcionalidades Clave:

Contacto con los objetos del juego y la realidad mediante el HTC Vive.

Wknbuy

Venta de Cupones

Wknbuy es una app desarrollada para iOS y Android con un administrador web.

Permite al administrador, desde su computadora o teléfono, registrar el precio, cantidad y puntos que una promoción tiene. Asimismo, puede determinar cuándo abre o cierra una venta, por ciudad. El usuario puede comprar las promociones mediante pagos con tarjeta o con puntos.

Funcionalidades Clave:

Ventas basadas en horarios, uso de multimedia, notificaciones, base de datos, pasarela de pagos.

Mr. Pampas

Información del restaurante

Mr. Pampas es una app desarrollada para iOS y Android.

Esta app ofrece la visualización del menú del restaurante, e incluye la ubicación y contacto de todos los establecimientos que hay en México. La app también incluye una tarjeta de visitas para el cliente frecuente, la cual al juntar tres visitas otorga un premio especial al cliente.

Funcionalidades Clave:

Notificaciones, Google Maps, Tarjeta de cliente frecuente.

Wknfly

Reservaciones

WKNFLY es una app desarrollada para iOS y Android.

Wknfly resuelve el problema de que te salten en la fila del restaurante, permitiendo al usuario generar reservaciones. La aplicación de hostess permite enviar una notificación al teléfono del cliente cuando sea su turno. Esta aplicación acepta pagos cuando el restaurante lo requiera.

Funcionalidades Clave:

Base de datos en tiempo real, pasarela de pagos, aplicacion de hostess y de cliente, localización en tiempo real.
